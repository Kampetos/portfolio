Laptop Store

This project is about a laptop store where customers can buy laptops and the administrator manages the laptops using a Content Management System (CMS). Built using HTML, CSS, JavaScript, PHP, and MySQL, and libraries like jQuery and Bootstrap.

Honda Genuine Parts

This project involves selling motorcycle spare parts such as oil, radiator covers, keys, etc. Has a Content Management System (CMS) for the administrator to manage the parts. Built using Laravel, with MySQL as the database.
